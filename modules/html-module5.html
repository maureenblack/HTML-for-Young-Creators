Module 5: Lists & Tables

Understanding Lists and Tables

In this module, we will learn how to create lists and tables in HTML, using Mount Eloumdem as an example to organize information.

To create an unordered list of features of Mount Eloumdem, use the following syntax:

<ul>
    <li>Stunning views</li>
    <li>Hiking trails</li>
    <li>Photography opportunities</li>
</ul>

For a table displaying visitor information, you can use:

<table>
    <tr>
        <th>Feature</th>
        <th>Details</th>
    </tr>
    <tr>
        <td>Height</td>
        <td>2,500 meters</td>
    </tr>
</table>

Exercise

Your task is to create an unordered list of activities you can do at Mount Eloumdem:

<ul>
    <li>Hiking</li>
    <li>Camping</li>
    <li>Bird watching</li>
</ul>

Make sure to add at least three items to your list.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
