What is CSS? ü§î

CSS stands for Cascading Style Sheets. Think of it as our artistic toolkit:

  • Cascading: Like water flowing down Mount Eloumdem, styles flow from general to specific
  • Style: Just as the mountain has different colors and textures, CSS adds visual style to our webpage
  • Sheets: We organize our styles in sheets, like organizing different areas of the mountain

Basic CSS Properties üéØ

Let's learn some basic CSS properties to style our mountain webpage:

Colors and Backgrounds

color: green;           /* Text color */

border: 2px solid brown;

Text Styling

font-size: 24px;       /* Text size */
font-weight: bold;     /* Bold text */
text-align: center;    /* Center alignment */

Spacing and Layout

margin: 20px;          /* Space around elements */
padding: 15px;         /* Space inside elements */
border-radius: 10px;   /* Rounded corners */
