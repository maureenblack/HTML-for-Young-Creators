Understanding HTML Elements

HTML elements are the building blocks of every webpage. In this module, we'll dive deep into how elements work and how to structure them properly.

In this module, you'll learn:

  • HTML document structure and hierarchy
  • Block vs. inline elements
  • Nesting elements properly
  • HTML5 semantic elements
  • Common attributes and their usage

1. Basic HTML Document Structure

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Webpage</title> </head> <body> <!-- Your content goes here --> </body> </html>

Each part serves a specific purpose:

  • <!DOCTYPE html>: Declares the document type and version
  • <html>: The root element of the page
  • <head>: Contains metadata about the document
  • <body>: Contains the visible content

2. Block vs. Inline Elements

HTML elements are typically displayed in two ways:

Block Elements

<div>Block element takes full width</div> <p>Paragraphs are block elements</p> <h1>Headings are block elements</h1>

Inline Elements

<span>Inline elements</span> flow with text <a href="#">Links are inline</a> <strong>Bold text is inline</strong>

3. Element Attributes

Attributes provide additional information or functionality to elements:

<img src="image.jpg" alt="Description" width="300" height="200"> <a href="https://example.com" target="_blank">Link</a> <div class="container" id="main">Content</div>

4. Nesting Elements

Elements can be nested inside other elements to create complex structures:

<div class="container"> <h1>Main Heading</h1> <p>This is a paragraph with <strong>bold text</strong> and a <a href="#">link</a> inside it.</p> </div>
