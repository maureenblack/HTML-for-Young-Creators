Module 2: HTML Elements & Structure

45 minutes Beginner

Understanding HTML Elements

HTML elements are the building blocks of every webpage. Let's explore how they work together to create structured content.

In this module, you'll learn:

  • HTML document structure and hierarchy
  • Block vs. inline elements
  • Nesting elements properly
  • Common HTML elements and their uses

HTML Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Webpage</title>
</head>
<body>
    <header>
        <h1>Welcome to My Site</h1>
        <nav>
            <ul>
                <li><a href="#">Home</a></li>
                <li><a href="#">About</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <article>
            <h2>Main Content</h2>
            <p>This is the main content of the page.</p>
        </article>
    </main>
    <footer>
        <p>© 2024 My Website</p>
    </footer>
</body>
</html>

Practice Exercise

Create a simple webpage that includes a header, navigation menu, main content area, and footer. Use the structure shown in the example above as a guide.
