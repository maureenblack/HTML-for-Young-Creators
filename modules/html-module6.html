Module 6: Forms & Input

Understanding Forms and Input

This module covers how to create forms for user input, using Mount Eloumdem as a context for collecting visitor feedback.

To create a simple feedback form for visitors to Mount Eloumdem, use the following syntax:

<form>
    <input type="text" placeholder="Your Name">
    <input type="email" placeholder="Your Email">
    <input type="submit" value="Submit">
</form>

In this example, the form collects a user's name and email address.

Exercise

Your task is to create a form that collects feedback about Mount Eloumdem:

<form>
    <input type="text" placeholder="Your Feedback">
    <input type="submit" value="Submit">
</form>

Ensure that your form is correctly formatted and includes a placeholder.
