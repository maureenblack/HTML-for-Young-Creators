Module 7: Semantic HTML

Understanding Semantic HTML

This module introduces semantic HTML, which uses HTML markup that conveys meaning about the content, with examples related to Mount Eloumdem.

Semantic elements include tags like <article>, <section>, and <footer>. For instance, you can use:

<article>
    <h2>Visit Mount Eloumdem</h2>
    <p>Mount Eloumdem is a beautiful destination for hikers and nature lovers.</p>
</article>

Exercise

Your task is to create a semantic section for a blog post about Mount Eloumdem:

<section>
    <h2>Exploring Mount Eloumdem</h2>
    <p>This section contains information about the various trails available at Mount Eloumdem.</p>
</section>

Ensure that your section is correctly formatted and includes a title.
