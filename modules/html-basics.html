Module 1: Introduction to HTML

30 minutes Beginner

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of web pages using markup.

In this module, you'll learn:

  • Basic structure of an HTML document
  • How to create your first HTML page
  • Understanding HTML tags and elements
  • Essential HTML document setup

Your First HTML Document

<!DOCTYPE html>
<html>
<head>
    <title>My First Web Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
</body>
</html>
